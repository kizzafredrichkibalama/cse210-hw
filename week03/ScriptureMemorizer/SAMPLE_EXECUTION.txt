KEY FEATURES DEMONSTRATED
----------------------------------------------------

1. REFERENCE HANDLING
   - Single verse: "John 3:16"
   - Multiple verses: "Proverbs 3:5-6"

2. WORD HIDING
   - Each word replaced with underscores matching word length
   - "God" → "___"
   - "so" → "__"
   - "believers" → "_________"

3. RANDOM SELECTION
   - Each run selects different scripture from library
   - Each press hides different random words (3-5 words)
   - No two runs are identical

4. PROPER TERMINATION
   - Press enter to hide more words
   - Type "quit" to exit
   - Program ends when all words hidden
   - Program ends when user types "quit"

5. CLEAN DISPLAY
   - Console cleared before each display
   - Reference always shown at top
   - Scripture text displayed below
   - User prompt at bottom
